简介

⚡ table-xlsx 基于SheetJS封装

⚡ 能够帮助你快速将xlsx文件转换成表格数据或表格数据导出生成xlsx文件

⚡ 导出支持简单样式设置、多sheet页、行/列合并等

note

⚡️ SheetJS/js-xlsx社区版本不支持样式，您可以使用@pengchen/xlsx(基于js-xlsx@0.17.0修改)
💸 当然，您也可以直接使用SheetJS Pro版本
